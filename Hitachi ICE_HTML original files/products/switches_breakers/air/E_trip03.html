Air Circuit Breakers : Electronic trip relay / OCR-III

External configuration

1 : Display LCD

  • Display of normal condition: In use current of R, S, T, N phases
  • Fault display: Fault phases and kinds of fault
    -Long-time delay trip: L_INV
    -Short-time delay trip: S_INV
    -Instantaneous-time delay trip: INST
    -Ground fault trip: GND →LCD reset method: Press "Func" button after "Ent"

EX)

FAULT : R
TYPE : L_INV

→ Long-time delay trip occurring in"R" phase

2 : Condition display LED : "Run"

  • The LED blinks with normal operation of OCR.

3 : Over-current display LED

  • "pick-up"LED : when over default pre-alarm current flows the "pick-up" LED warning blinks and it is turned off after OCR operation.
  • "Fault" LED : The OCR operation of generating fault turns on "Fault" LED.

4 : Communication display LED : "Rx, Tx"

  • Blinks with normal communication of OCR.

5 : Reset button

  • Out-put contact signal reset and OCR over-current display LED.

6 : Selection button

  • Func : Mode selection button
  • Ent : Selection button of various set-up value
  • , : Set-up value input button

7 : RS232 port

: OCR operation check port as connecting PC

8 : TEST Port

: Connection button with OCR tester (OT-2000)

It is tested by IEC standard 61000-4-3 for mobile checking function and certified class"A" of frequency range from 80MHz to 2.4GHz.
