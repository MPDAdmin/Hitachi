Remote Monitoring System (COSMOSII)

Web monitoring system shows real time status of compressors via office computer with high speed interface (100BASE-T).

  • Features
  • Specifications
  • Brochures & Enquiry

Features

Labour saving

A COSMOSII module is capable of setting and monitoring operating conditions of up to four compressors.

Monitoring energy saving

A COSMOSII module can monitor the history of compressor load from the data of load factor, ampere, mean-load and other operating data.

Immediate failure notice

Operating conditions can be monitored visually by animations and bar charts. In an emergency, the operating data and shutdown history are conveyed immediately to make necessary maintenance.

Easy Installation

FiS485 Multi Drop cable system is applied. In addition, connecting to existing LAN cable makes wiring easier and economical.
