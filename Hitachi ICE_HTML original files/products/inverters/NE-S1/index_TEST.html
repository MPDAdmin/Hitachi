NE-S1 Series

Model Nomenclature


The NE-S1 Series is economical with a simple operating system. Popular for use with 0.2kW to 4.0kW. Designed with an ultra-compact model for integration in most applications.

  • Features
  • Specifications
  • Brochures & Enquiry

Features

  • Among the smallest form-factors in their category
  • 43% smaller than equivalent X200 (0.2 kW)
  • Side-by-side installation to save panel space
  • Run/Stop/Reset is integrated in one button for simple operation
  • Full-function attachable operator available as an option

  • Logic input is compatible with both sink and source logic
  • Wide input power voltage range
  • RS-485 Modbus® RTU Communication port
  • Optimal performance for energy saving applications such as fans and pumps.
