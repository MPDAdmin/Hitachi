Suspension Type Crane Saddle

Features

  • As double channel method frame is adopted, the suspension type crane saddle is highly reliable and durable.
  • A motor with break is adopted, and the brake torque can be adjusted
  • Manufacturing time of the crane can be reduced by the centering mark for adjustment.
